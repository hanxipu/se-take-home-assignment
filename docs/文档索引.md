- 后端：**Golang**；前端 UI：**Angular**。
- **语言**：仓库根目录 [README.md](../README.md)、Angular **界面文案**、CLI **对用户可见**的提示与演示日志等，使用 **英文**（与作业描述一致）。**`docs/` 下设计文档** 与 **源码注释**（含 Go、TypeScript、脚本内说明性注释）使用 **中文**。
- 代码实现阶段要求：简洁、可复用。
- 作业原文见仓库根目录 [README.md](../README.md)。
